# NiveLineWeb

NiveLine için proje kökünde oluşturulmuş bağımsız statik landing page.

## Dosyalar

- `index.html`: ana landing page
- `styles.css`: tüm görsel sistem ve responsive düzen
- `script.js`: scroll reveal ve küçük etkileşimler
- `assets/brand-mark.svg`: marka işareti
- `assets/favicon.svg`: favicon

## Lokal çalıştırma

```bash
python3 -m http.server 8080 --directory NiveLineWeb
```

Sonra tarayıcıda:

```text
http://localhost:8080
```

## Sonraki mantıklı adımlar

- gerçek App Store ve Google Play bağlantılarını eklemek
- `support@niveline.com` ve benzeri adresleri üretim adresleriyle netleştirmek
